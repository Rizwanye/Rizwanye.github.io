Enhancing XAI Interpretability: Using Radar Charts to Simplify Feature Based Explanations and Reduce Cognitive Load (Honours's Thesis) Star Fork

My honours research focuses within the field of Explainable Artifical Intelligence (XAI) and Visual Explainable Artifical Intelligence (vXAI). This project was based on a hypothetical client "XAI Bank". As most ML models are black boxes and difficult to understand especially in the banking industry. This project focuses on improving interpretability and explainability through the use of a tailor made radar chart combined with a large language model.

The web-app was able to generate automatic textual explanations for the end user combined with a radar chart to help explain the loan approval outcome.

Military Object Detection Project Built using Python (Undergraduate Project) Star Fork

This project utilised Python programming, specifically the yolov5 package to detect military assets such as soldiers, supply trucks and assult vehicles. It is trained on infra-red imagery of simulated objects from drone video footage. And launched as a web-application using flask.

This entire project was built by myself, from the dataset creation, to training the image classifier, which demonstrates my ability to innovate. The main goal of this project was to demonstrate how object detection can be useful in intelligence gathering and automation in the military context.
